# lottery-game-dApp
It is a lottery game in which the maintainer (owner of contract) will start the game and others can join. When the maximum player limit is reached, 
the winner is chosen using Chainlink VRF (Verifiable Random Function).

### **Deployed Link:**
https://lottery-game-d-app.vercel.app/

### **Tech Stacks used:**
- NextJs
- Solidity + Hardhat
- Chainlink VRF
- The Graph
